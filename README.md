Python程式設計 - 期末專題

主題發想：讓日常查詢天氣更便利化

解決方案：用api取得每天及5天的天氣資訊，並且利用socket傳輸到IOT裝置，可以放在隨時可看到的位置增加生活便利性。

延伸構思：程式每24小時取得一次資訊並更新IOT顯示資訊，外加在OLED實時更新時間，依天氣或溫度創建圖示或文字提示，例如:太陽符號、下雨符號、雲符號、熱、冷...等等。

天氣api網站:https://home.openweathermap.org/
